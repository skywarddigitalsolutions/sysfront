# Reglas de Desarrollo del Proyecto (Persistente)
⚠️ IMPORTANTE:
- El desarrollo debe realizarse por **subtareas**.
- Después de cada subtarea debes detenerte y preguntar si deseo continuar.
- NO avances sin mi confirmación explícita.
- Cada función generada debe estar comentada en español, indicando:
  - Breve descripción.
  - Parámetros: nombre, tipo, breve descripción (<5 palabras).
  - Retorno: nombre, tipo, breve descripción (<5 palabras).

  Ejemplo de comentario:
  ```typescript
  /**
   * Breve descripción.
   * @param param1 Breve descripción (<5 palabras).
   * @param param2 Breve descripción (<5 palabras).
   * @returns Breve descripción (<5 palabras).
   */
  ```
